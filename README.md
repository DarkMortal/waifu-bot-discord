# Waifu Discord Bot
A Discord bot to fetch random waifu images! 
## Commands

`/waifu` - Get Random Waifu Image
 &nbsp;
-`/neko` - Get Random Neko Image
 &nbsp;
-`/shinobu` - Get Random Shinobu Image
 &nbsp;
-`/megumin` - Get Random Megumin Image


## Config Vars
`TOKEN` - Your Discord bot token
